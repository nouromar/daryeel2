# Code of Conduct

We are committed to providing a welcoming and professional environment for everyone who participates in this project.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and communicate clearly.
- Focus on what’s best for the project and users.

## Unacceptable behavior

- Harassment, discrimination, or hateful conduct.
- Personal attacks or sustained disruptive behavior.
- Sharing private information without consent.

## Reporting

If you experience or witness behavior that violates this code of conduct, please report it to the maintainers.
